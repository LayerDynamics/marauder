## Key Architecture Decisions

1. **Overlay rendering**: Use instanced quads with alpha blending, same pattern as bg/text passes
2. **Mouse → cell mapping**: `cellCol = floor(mouseX / cellWidth)`, `cellRow = floor(mouseY / cellHeight)`
3. **Scrollback rendering**: Renderer reads from grid with viewport offset, grid provides row data from scrollback ring buffer
4. **URL click**: Ctrl/Cmd+Click pattern (standard terminal convention), not bare click (which interferes with selection)
5. **Clipboard**: Use Tauri plugin (not raw FFI) for cross-platform clipboard access
